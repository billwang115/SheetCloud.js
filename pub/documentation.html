API Documentation

SheetCloud is a javascript library that provides a music sheet interface for websites. The SheetCloud API allows for interacting with the Sheetcloud library.

Getting Started

To setup SheetCloud, ensure the library is in the scope of your project. Then, add these script lines to your html file:

<script defer type="text/javascript" src="js/SheetCloud.js"></script>
<link rel="stylesheet" type="text/css" href="js/SheetCloud.css" />

Once the library is imported, a music sheet can be created using these library calls:

const sg = SheetGenerator();
sg.makeSheet({ clefs: { treble: true, bass: true }, timeSignatures: { upper: 2, lower: 4 }, numStaffs: 1, tempo: 120, items: [] });
const sheetHTMLElement = sg.mainView;    //sg.mainView is the html music sheet that can be inserted into your html page

Methods

These are methods that can be accessed from the object returned sheetGenerator object returned from the library.

.makeSheet()

initializes the sheet html element and sets the mainView property of the sheetGenerator object to this element.

Parameters
  • options : Object corresponding to configuration options
    • clefs: Object corresponding to which clefs to show
      • treble: bool
      • bass: bool
    • timeSignatures: Object corresponding to the timesignatures to display
      • upper: number
      • lower: number
    • numStaffs: Number corresponding to the number of rows in the sheet to show
    • tempo: Number corresponding to the speed that the sheet should be played at
    • items: Array of objects corresponding to a note to be displayed on the music sheet
      • type: String corresponding to the type of the note
      • positionLeft: String corresponding to the distance of the note from the left section of a staff
      • positionTop: String corresponding to the distance of the note from the top of the staff
      • staffIndex: Number corresponding to the index of the staff that this note belongs in
      • grandStaffIndex: Number corresponding to the index of the grandStaff that this note belongs in.
Return Value
  • None

.enableEditing()

Enable Editing for the Music sheet and display editing tools

Parameters
  • enabled: Bool
Return Value
  • None

.saveSheet()

Save configuration of notes/utilities and sheet settings and return configuration

Parameters
  • None
Return Value

      • bass: bool
    • timeSignatures: Object corresponding to the timesignatures to display

      • lower: number

    • items: Array of objects corresponding to a note to be displayed on the music sheet
